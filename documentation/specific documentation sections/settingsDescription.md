            2.1. Promenu teme kalendara
            2.2. Isključivanje/Uključivanje obaveštenja
            2.3. Noćni režim

        3. Korisnik bira željenu opciju za podešavanje.

        4. Podešavanja se primenjuju i čuvaju.
